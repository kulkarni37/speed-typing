# ⌨️ Speed Typing Lite

A simple and interactive typing speed game built using **HTML**, **CSS (Bootstrap 4)**, and **JavaScript**. Test your typing speed by typing randomly displayed words before the timer runs out.

---

## 🚀 Features

* 🎯 Random word generation
* ⏱️ Countdown timer
* 📈 Live score tracking
* 🎮 Three difficulty levels

  * Easy (5 seconds)
  * Medium (3 seconds)
  * Hard (2 seconds)
* ✅ Instant correctness feedback
* 📱 Responsive design using Bootstrap
* 🔄 Continuous gameplay

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/kulkarni37/Speed-Typing-Lite.git
```

2. Open the project folder.

3. Open `index.html` in your web browser or run it using the **Live Server** extension in Visual Studio Code.

No additional installation or dependencies are required.

---

## 🎮 How to Play

1. Launch the game.
2. Select your preferred difficulty level.
3. Type the displayed word before the timer reaches zero.
4. Every correct word:

   * Increases your score.
   * Resets the timer.
   * Displays a new random word.
5. If the timer reaches zero, the game ends and your score resets.
